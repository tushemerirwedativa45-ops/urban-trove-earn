🏦 How the Deposit System Works

  • Users choose a plan: UGX 30,000 → 40,000 | 40,000 → 50,000 | 50,000 → 60,000 | 60,000 → 70,000
  • The Airtel account number is hardcoded as 0702762675 — only that number is accepted
  • After deposit, the system waits 5 seconds then automatically adds the interest return to the user's balance
  • All data is stored in the user's browser localStorage — there is no real server or database yet
  • This means each user's data only exists on their own device/browser

🎮 Urban Jet Game — How It Works

  • The crash point is taken from a fixed sequence: [1.1, 3, 1.40, 1.08, 2.5, 1.15, 4.2, 1.05, 2.8, 1.20] then loops
  • A random value of up to +0.3 is added to each crash point to make it feel unpredictable
  • The multiplier increases by 0.003 per animation frame (~60fps)
  • Game balance starts at UGX 1,000 (virtual, not real money)
  • Players can place 2 bets simultaneously and cash out independently
  • If a player does not cash out before the crash, the bet is lost
